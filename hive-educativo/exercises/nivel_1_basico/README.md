# Nivel 1: Básico - HiveQL

Aprende los fundamentos de HiveQL:

## Temas Cubiertos
- CREATE EXTERNAL TABLE
- SELECT y LIMIT
- WHERE (filtrado)
- GROUP BY y agregaciones (COUNT, SUM, AVG)
- ORDER BY

## Ejercicios
1. **Consulta SELECT Básica** - Cargar y consultar datos
2. **Filtrado con WHERE** - Filtrar registros
3. **Contar Registros** - Agregación con COUNT

💡 Cada ejercicio incluye pistas y solución de referencia.
